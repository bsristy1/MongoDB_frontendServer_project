# MongoDB_frontendServer_project

App Description: Allows user to fetch jokes based on category and store/view favorite

APIs: Jokes(https://v2.jokeapi.dev)

* **Dynamic Joke Fetching:** Retrieve random jokes from the **JokeAPI**.
* **Category Filtering:** Select specific categories to customize the humor (Programming, Pun, Spooky, Christmas, Misc).
* **Safe Mode:** Content is automatically filtered to exclude NSFW/racist/sexist/explicit content.
* **Favorites System:** Sta" jokes to save them permanently to a MongoDB database.
* **Favorites View:** View a list of all previously saved jokes.
* **Responsive Design:** UI styled with CSS.
